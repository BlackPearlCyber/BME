IoT Based Wearable Patient Monitoring

Demo and talking points for presentation — live demo embedded on the right.

Abstract (short)

This wearable system monitors heart rate, SpO2, body temperature and fall events. Data flows from a MAX30102 on Arduino Nano to an ESP32 which handles WiFi and Telegram alerts.

Key Features

  • Heart rate & SpO2 via MAX30102
  • Temperature via DS18B20
  • Fall detection via MPU6050
  • Emergency SOS button and Telegram alerts

Wiring Summary

  • MAX30102: I2C to Arduino A4(A) / A5(SCL), VIN 5V, GND
  • Arduino TX -> ESP32 RX2(GPIO16), common GND
  • DS18B20 DATA -> ESP32 GPIO4 + 4.7k pull-up
  • MPU6050: I2C to ESP32 GPIO21/22
  • SOS button -> GPIO15 (pulled-up)
